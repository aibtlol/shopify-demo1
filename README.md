# Shopify Custom Theme Demo

Live Preview: http://127.0.0.1:9292/

This is a technical demo for Shopify Online Store 2.0 theme development, showing real implementation of custom Liquid sections, frontend interactions and modern CSS build pipeline.

## Tech Stack
- Shopify Liquid (OS2.0 Section / Schema / Blocks)
- Tailwind CSS v4 (Vite bundled & compressed)
- Alpine.js for lightweight frontend interactivity
- Vite for CSS build & output automation
- Shopify CLI theme deployment workflow
